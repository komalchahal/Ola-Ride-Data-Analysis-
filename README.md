#Ola Ride Data Analysis

Project Overview

This project analyzes Ola ride data to identify patterns in ride trends, customer behaviour, and driver performance. The analysis was conducted using SQL for data extraction and Power BI for data visualization.

Tools & Technologies
- SQL
- Power BI
- Data Visualization

Key Analysis

- Identified successful and cancelled bookings
- Calculated average ride distance by vehicle type
- Analyzed reasons for ride cancellations by customers and drivers
- Computed metrics such as driver ratings, completed rides, and total booking value

Dashboard Insights

Power BI dashboards were created to visualize:

- Ride volume trends over time
- Booking status distribution
- Revenue by payment method
- Customer ratings by vehicle type
- Top customers and most popular vehicle types

Conclusion

The analysis provided insights into ride demand, customer preferences, and driver performance. These insights can help improve operational efficiency and enhance decision-making for ride-hailing services.
